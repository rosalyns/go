Algemeen:

Bord: linksboven is 0,0
Volgorde van argumenten: x,y dus eerst rij en dan kolom (LET OP GUI)
Default port: 5647
TIMEOUTLIMIT: 1.5 minuut


Client naar server:

EERSTE:
NAME <clientnaam> VERSION <int> EXTENSIONS 0 0 1 1 0 0 0 (Volgorde: chat challenge leaderboard security 2+ simultaneous multiplemoves)


MOVE <int>_<int> (als je wil passen MOVE PASS)
SETTINGS <kleur> <bordgrootte>
QUIT (midden in spel stoppen)
REQUESTGAME <aantal spelers> <naam tegenspeler> (RANDOM als je geen challenge doet)
ACCEPTGAME <ander>
DECLINEGAME <ander>
LOBBY

Extensions:
CHAT <bericht>
LEADERBOARD

Server naar client:
EERSTE:
NAME <servernaam> VERSION <int> EXTENSIONS 0 0 1 1 0 0 0 (Volgorde: chat challenge leaderboard security 2+ simultaneous multiplemoves)


START <aantal spelers> (naar 1ste speler)
START <kleur> <bordgrootte> (naar 2de speler)
TURN <huidige speler> <int>_<int> <volgende speler> (voor de eerste beurt FIRST sturen, als INVALID is huidige speler = volgende speler)
ENDGAME <reden> <winnende speler> <score> <verliezende speler> <score> (reden kan zijn: FINISHED, ABORTED, TIMEOUT)
REQUESTGAME <vuitdager>
DECLINED <ander> (alleen naar uitdager en uitgedaagde)
LOBBY <naam1>_<naam2> (alle vrije spelers)


Extensions:
CHAT <naam> <bericht>
LEADERBOARD <naam> <score> (x10) 


Errors:
ERROR <type> <bericht> (bv UNKNOWN COMMAND, INVALIDMOVE, NAMEEXISTS)

DELIMITER = "$" (in plaats van alle spaties hier)
DELIMITER2 = "_"
